# Portfolio (TypeScript)

This is a modern developer portfolio built with React, Vite, Tailwind CSS, and TypeScript. It features advanced visuals, professional layout, and multiple sections (About, Projects, Skills, Experience, Education, Testimonials, Blog, Open Source, Awards, Tech Stack, Hobbies, Contact, and more).

## Getting Started

1. **Install dependencies:**
   ```
   npm install
   ```
2. **Start the development server:**
   ```
   npm run dev
   ```
3. **Build for production:**
   ```
   npm run build
   ```

## Features
- TypeScript-first React components
- Vite for fast development
- Tailwind CSS for styling
- Animated transitions, parallax, SVG backgrounds, and particle effects
- Responsive and accessible design
- Theme toggle (light/dark)
- All major developer portfolio sections

## Customization
- Replace content in the `src` directory with your own information.
- Update the avatar image and resume in the `public` folder.
- Edit Tailwind config for custom styles.
